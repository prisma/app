# Inspiration library

This directory is a **living set of notes** on systems we’re using as reference points for the Prisma App Framework.

The goal is to **capture conclusions once** (and refine them over time) so we don’t re-derive the same comparisons in every design discussion.

## How to use these docs

Each document should answer:

- **Similarities**: which elements of their domain map to ours?
- **Inspirations**: what do we want to emulate directly vs adapt?
- **Relevance**: why is this system a good reference point for this part of the Prisma App Framework?
- **Open questions**: what should we research next, and what assumptions might be wrong?
