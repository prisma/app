# Templates

Use these templates when adding new documentation. Keep them consistent so the docs remain easy to scan and evolve.

- `north-star.md`: goals/purpose docs
- `guiding-principles.md`: guiding principles
- `architectural-principles.md`: architectural principles
- `example-app.md`: example app anchor docs
- `domain.md`: bounded-context / domain deep dive
- `adr.md`: Architecture Decision Record
